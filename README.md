# geo_distance_calculator
A location calculator i have made in my CIS 162 class.

## Overview
A command-line Python application that processes a dataset of 41,000+ U.S. towns to calculate distances between locations using coordinate-based mathematical formulas.

## Features
- Zip code and town lookup
- Spreadsheet data parsing
- Distance calculation using coordinate math
- Command-line interface

## Technologies Used
- Python
- CSV/Spreadsheet data processing

## How to Run
- Input info into zipinfo.py
- run to get output
